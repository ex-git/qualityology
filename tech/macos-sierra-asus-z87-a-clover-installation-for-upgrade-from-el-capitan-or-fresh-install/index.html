macOS Sierra Asus Z87-A Clover Installation for Upgrade from El Capitan Or Fresh Install

Oct 13 2016 tech 3 minutes read (About 436 words)

Instruction for how to prepare Clover USB system installation flash driver can be found from here: Install Mac OS X with clover EFI bootloader on ASUS Z87a Hardwares used in this instruction: Motherboard: Asus Z87-A motherboard (Get ASUS Z87-A motherboard from Amazon) CPU: I7 4770K CPU or any LGA1150 CPU (Get LGA1150 CPU from Amazon) Graphics Card: Integrated Graphics Processor (HD Graphics 4600) Sound Card: Integrated Realtek ALC892 Harddriver: SATA3 SSD Clover’s configuration and files from this guide will fit El Capitan and Sierra. You can directly upgrade from previous macOS to Sierra using this guide for Asus Z87-A motherboard or other similar Asus motherboards. What is working? CPU Power management/Sleep/Wake OK USB 3.0 OK USB 2.0 OK ALC892 Sound Card OK Graphic Card OK Almost everything What is not working? None macOS bug or Clover bug and fix?

  • WiFi issue after wake from sleep? uncheck Disable Wake for network access from Energy Saver and restart
  • No sound after wake from sleep? Install CodecCommander, download the latest version from here: https://bitbucket.org/RehabMan/os-x-eapd-codec-commander/downloads
  • iMessage issue? Do you have 17 digit MLB# and Board Serial number? 15 digit Serial#? and SmUUID?

  1. Clover configuration

  1. download SSDT-XOSI.aml and ssdt.amland unzip to EFI/CLOVER/ACPI/patched

  2. download AppleALC.kext FakeSMC.kext RealtekRTL8111.kext USB_Series8_Injector.kext and unzip to EFI/CLOVER/kexts/Other

  3. download FSInject-64.efi HFSPlus.efi NTFS.efi OsxAptioFix2Drv-64.efi OsxFatBinaryDrv-64.efi and unzip to EFI/CLOVER/drivers64UEFI Delete other files in this folder.

  4. upgrade to macOS Sierra or do a fresh installation.
